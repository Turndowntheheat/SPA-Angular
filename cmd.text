expose a component:
# In mfe1/src/app
ng generate component mfe1-component

Edit webpack.config.js to expose it:
exposes: {
  './Mfe1Component': './src/app/mfe1-component/mfe1-component.component.ts',
},

Repeat similarly in mfe2.
